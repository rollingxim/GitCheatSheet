Команды Git
Команда Описание
git init создание нового репозитория
git clone url клонировать папку к себе на компьютер
git add file выбрать один файл, чтобы сохранить изменения в нём
git add . выбрать сразу все измененныей файлы для сохранения
git commit -m "comment" сохранить изменения и подписать, что было сделано
git commit -a -m "comment" быстрое сохранение, найдутся все измененные файлы и сохранятся одной командой
git push url отправить сохраненные изменения в интернет, на сервер
git status проверить, какие файлы изменены и какие ещё не сохранены
git log посмотреть все предыдущие сохранения
git branch branch_name создать ветку, чтобы выполнять новые задачи, не трогая основной проект
git checkout branch_name перейти на другу ветку
git checkout -b branch_name -
git remote -
